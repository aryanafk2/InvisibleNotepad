# Ghost Notepad

A lightweight, semi-transparent, dark-themed Notepad built with C++ and the Win32 API.

## Features

- **Adjustable transparency:** Control window opacity.
- **Toolbar controls:** Quickly change transparency, toggle minimal view, and always-on-top mode.
- **Dark theme:** Greyish-black background with white text for comfortable coding or note-taking.
- **Resizable window:** Drag edges or corners, with a minimum size showing only the toolbar.
- **Always-on-top toggle:** Keep the notepad above other windows.
- **Minimal view mode:** Collapse the window to show only the toolbar.
- **Close & minimize buttons:** Standard window operations with custom buttons.
